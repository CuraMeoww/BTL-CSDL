## 📂 Thành phần Báo cáo
Hệ thống lưu trữ đầy đủ các thành phần phục vụ đánh giá BTL bao gồm:

### 1. Tài liệu báo cáo & Thiết kế
*   `Báo cáo BTL Quản Lý Cửa Hàng.pdf`: Nội dung chi tiết quá trình phân tích và xây dựng hệ thống.
*   `Sơ đồ ER.pdf`: Sơ đồ thực thể liên kết ER.
*   `Mô hình quan hệ.pdf`: Mô hình dữ liệu quan hệ sau khi chuẩn hóa.

### 2. Cơ sở dữ liệu (SQL)
*   `QuanLyCuaHangQuanAo.sql`: Script khởi tạo cấu trúc 15 bảng và chèn dữ liệu mẫu.
*   Thư mục `/Truy vấn SQL`: Chứa 16 câu lệnh truy vấn từ cơ bản đến nâng cao.

### 3. Mã nguồn Web Demo
*   `server.js`: Backend xử lý API bằng Node.js & SQLite3.
*   `index.html`: Frontend giao diện người dùng (Theme Cam đào - Hồng phấn).
*   `database.db`: File dữ liệu SQLite hiện tại.
